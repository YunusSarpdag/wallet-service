  ## Beginning
  Requires Java 17

  ## Useage
  Clone the repository
  git clone https://github.com/YunusSarpdag/wallet-service
  cd wallet-service
  install maven
  mvn clean install
  mvn spring-boot:run

  if you have IDE you can run in main method
